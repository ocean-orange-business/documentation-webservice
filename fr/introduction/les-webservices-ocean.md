# Les Web services Océan

### Introduction

**Océan, une filiale de Shiftmove, propose aux entreprises disposant d’un parc de véhicules (VL, VUL, engins) et d’objets (matériels) des solutions innovantes de gestion des activités mobiles, avec ou sans fonctionnalités de géolocalisation.**

Une plateforme technologique avancée, intégrant l’intelligence artificielle et le cloud, est mise à disposition des clients, offrant un webservice exposant l’ensemble des web services disponibles. 
Ces web services permettent aux utilisateurs d’ajouter, de modifier, de supprimer ou de consulter leurs données de manière simple et sécurisée. 
Une documentation détaillée de chaque web service, incluant ses spécificités techniques et ses prérequis, est fournie selon le schéma suivant :

1.  Description
2.  Web services concernés, avec leurs spécificités techniques
